◆ ID_Utente()

int ID_Utente ( )

Includo le librerie necessarie per il corretto funzionamento del programma.

Acquisisce l'ultimo numero ID tra tutti gli utenti, posizionandosi all'ultima posizione e immagazzinando in una variabile il suddetto ID.

Precondizione
La funzione, per funzionare correttamente, ha bisogno che il file dei passeggeri sia correttamente caricato nel sistema.
Postcondizione
L'intero restituito sarà "ID", nel caso in cui il file esista. Verrà restituito 0 se non ci sono utenti, oppure se il file non esiste.

Struct per gli utenti, da cui prendere l'ID

Dichiarazione del file da cui pescare l'ultimo utente

Se il file non esiste, restituisci 0.

Variabile che conterrà l'ID

Mi posiziono all'ultima riga del file tramite SEEK_END, ma, volendo l'ultimo utente, lo leggo attraverso l'offset giusto "(-1)*sizeof(passeggero)"

Immagazzino l'ID dell'utente trovato in ultima posizione nella variabile "id".

Se l'ultimo ID è <= 0 (significa che l'ultimo utente è stato cancellato) allora, prendo l'ID del penultimo utente.

Mi sposto al penultimo utente

Riassegno il penultimo ID nella variabile "id"

Chiudo il file dei passeggeri

Ritorno alla funzione l'ID dell'utente trovato.

Ritorno 0 alla funzione se il file non è stato trovato.
